This is a full-stack web project inspired by Zomato, built using the MERN stack (MongoDB, Express, React, Node.js).  
The project is currently in progress, with the frontend layout completed using HTML, CSS, and JavaScript.

The main goal of this project is to replicate core features of Zomato such as restaurant listings, search functionality, user reviews, and order tracking.
